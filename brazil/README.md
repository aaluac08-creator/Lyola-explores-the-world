# The World of Brazil - Interactive Scrollytelling Website

An immersive, data-driven journey through Brazil's geography, culture, and complexity.

## Project Overview
- Interactive maps and data visualizations
- Scroll-driven animations and scrollytelling narratives
- Regional breakdowns with cultural insights
- Educational content presented with cinematic pacing

## Technology Stack
- **Scrollama.js** for scroll-driven interactions
- **D3.js** for maps and data visualizations
- **GSAP** for animation timelines and scroll triggers
- **Vanilla JavaScript** for application logic
- **CSS3** for layout and styling

## Project Structure
```
countries-webpage/
|-- index-brazil.html   # Main page
|-- css/                # Stylesheets
|-- js/                 # JavaScript modules
|-- assets/             # Images, fonts, and data
`-- README.md           # Project documentation
```

## Getting Started
### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional but recommended)

### Installation
1. Clone or download this repository.
2. Open `index-brazil.html` in your browser, or start a local server:

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server

# VS Code Live Server extension
```

3. Navigate to `http://localhost:8000`.

## Features by Section
1. **Hero** - Animated introduction
2. **Geography** - Interactive map of Brazil
3. **Regions** - Scroll-guided regional deep dives
4. **Demographics** - Population and ethnic breakdowns
5. **Government** - Political system and timeline
6. **Culture** - Cultural quirks and highlights
7. **Legacy** - Cultural reach and influence
8. **Hospitality** - Brazilian warmth and stories
9. **Outro** - Reflective wrap-up with calls to action

## Development Notes
- Each major section has dedicated CSS and JS modules.
- Utilities live in `js/utils.js` for shared helpers.
